# Project Handoff

Use when starting a new Claude Code session or transferring work.

Include:
- Repo purpose and stack.
- Current task and desired outcome.
- Important files and directories.
- Commands for setup, verification, and local development.
- Known issues, failing checks, or incomplete work.
- Next exact steps.

Avoid full logs and broad file dumps. Point the next session to CodeGraph/Serena first.
