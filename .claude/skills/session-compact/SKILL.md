# Session Compact

Use before `/compact` or when handing a long active session to a smaller context.

Preserve:
- Current goal and user constraints.
- Key decisions and assumptions.
- Files created, modified, or intentionally left alone.
- Commands run and important results.
- Failing tests, errors, or warnings.
- Next exact steps.
- Risks, blockers, and open questions.

Keep it short. Prefer bullets with filenames and commands over narrative.
